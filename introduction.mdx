# Welcome to Astro Services

Astro Services is a modern full-stack web application for showcasing and managing streaming and IPTV services. Built with cutting-edge technologies, it provides a seamless experience for browsing service offerings with real-time search, pricing, and discount information.

## What is Astro Services?

Astro Services is a complete SaaS platform that combines a beautiful React frontend with a robust Express API backend, all powered by MongoDB for flexible data management. The platform enables you to:

- **Browse Services** — View streaming and IPTV offerings in an elegant card-based grid layout
- **Search in Real-Time** — Find services instantly with dynamic filtering
- **Manage Content** — Full CRUD operations via REST API
- **Deploy Anywhere** — Docker-ready containerized architecture

## Architecture Highlights

The platform follows a clean three-tier architecture:

1. **Frontend Layer** — React SPA with component-based design
2. **API Layer** — RESTful Express server with route controllers
3. **Data Layer** — MongoDB with Mongoose schema validation

All components are containerized and can be deployed independently or together using Docker Compose.
